# Project Scope

## Legislative Scope

This project analyzes data from the Brazilian Federal Legislative Branch,
specifically the Chamber of Deputies (Câmara dos Deputados).

Data source:
https://dadosabertos.camara.leg.br/api/v2

## Scope Included

- Federal deputies
- Parliamentary fronts
- Legislative events
- Voting sessions
- CEAP expenses
- Parliamentary inquiry commissions (CPIs)

## Scope Excluded

- Senate data
- State legislatures
- Municipal legislatures
- Electoral court data
- Executive branch data

## Legislative Time Context

The project considers legislative periods (legislaturas)
available through the Chamber Open Data API.

Field:
leg_id_legislature

Example:
- 56th Legislature → 2019–2023
- 57th Legislature → 2023–2027
